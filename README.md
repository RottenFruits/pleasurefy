# pleasurefy

## How to run
- clone
```
git clone https://github.com/RottenFruits/pleasurefy.git
cd pleasurefy
```

- build and server run
```
docker build -t pleasurefy .
docker run --rm -p 8090:8000 pleasurefy
```

- access server
    - http://localhost:8090/
